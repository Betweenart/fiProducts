# Product list web app

Created with Angular, Bower, Grunt and Node for FaceIt.

## Basic useful feature list:

 * Get products list
 * Edit product name and price
 * Add new product
 * Remove product 
 * Sort ascending or descending by id, name or price


## Rest services:

 * get
 * get:id
 * post: add/create
 * delete: remove/delete
 * put: save/update
